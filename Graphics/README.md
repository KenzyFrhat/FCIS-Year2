# 🎨 Graphics Projects – README

## 📌 Overview

This repository contains coursework projects for the **Graphics** subject at university.
The folder is organized into two main subfolders, each representing a different project that explores visual design and rendering using different tools and approaches.

---

## 📂 Project Structure

```
Graphics/
│
├── Cartoon Character (Inkscape)

└── Angry Emoji (HTML Canvas + JavaScript)
```

---

## 🖌️ Project 1: Cartoon Character (Inkscape)

### 📖 Description

This project focuses on creating a **cartoon character illustration** using the vector graphics editor **Inkscape**.
The goal was to understand vector drawing concepts such as:

* Paths and shapes
* Layering
* Stroke and fill
* Bezier curves

### 🎯 Objective

* Practice vector-based drawing
* Improve precision in digital illustration
* Understand how complex shapes are constructed from simple primitives

### 📺 Reference Tutorial

The character was drawn by following this tutorial:
🔗 https://www.youtube.com/watch?v=PTc5JRYFy-A&t=706s

---

## 😠 Project 2: Angry Emoji (HTML Canvas + JavaScript)

### 📖 Description

This project implements an **angry emoji face expression** using the **HTML Canvas API** controlled by **JavaScript**.

### 🎯 Objective

* Understand coordinate systems in Canvas
* Learn how to draw shapes programmatically (circles, arcs, lines)
* Control facial expressions using geometry
* Translate visual ideas into mathematical drawing logic

### 🛠️ Technologies Used

* HTML5 Canvas
* JavaScript (Vanilla JS)

### 💡 Key Concepts Applied

* Drawing arcs for face and eyes
* Using paths for eyebrows and mouth
* Positioning elements relative to the canvas
* Creating expressions by adjusting angles and curves

---

## 🚀 How to Run

### For the Inkscape Project

1. Open the `.svg` file using **Inkscape**
2. Explore layers and paths used to build the character

### For the Canvas Project

1. Open the HTML file in any browser
2. Ensure JavaScript is enabled
3. The emoji will render automatically on page load
